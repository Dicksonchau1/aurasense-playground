# Handwash Module Feedback Collection Plan

## 1. Key User Groups
- Nursing students
- Clinical educators/instructors
- Infection control specialists
- Product team members

## 2. Demo & Walkthrough Checklist
- Access `/rehearse/handwash` from the main dashboard
- Start a new hand hygiene session
- Progress through all 7 steps, observing timer and feedback
- Complete the session and review the summary
- Add optional notes and view session history
- Test on both desktop and mobile
- Use only keyboard navigation to complete a session
- Try with screen reader enabled

## 3. Feedback Form Template
- **How clear was the step-by-step guidance?** (1–5)
- **Was the timer and progress display helpful?** (Yes/No/Comments)
- **Did you encounter any confusion or unclear instructions?** (Open)
- **Was the module accessible and usable with keyboard only?** (Yes/No/Comments)
- **Any issues with color contrast or visual clarity?** (Yes/No/Comments)
- **How would you rate the overall experience?** (1–5)
- **Additional suggestions or comments?** (Open)

## 4. Feedback Collection & Prioritization Process
- Distribute the module link and feedback form to all user groups
- Collect responses over a set review period (e.g., 1 week)
- Aggregate feedback and identify common themes or issues
- Prioritize changes based on frequency, severity, and impact
- Review prioritized list with product/clinical leads before implementation
